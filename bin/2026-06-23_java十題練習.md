## 第 1 題：三數中位數 ★

**題目**
輸入三個整數 `a`、`b`、`c`，回傳這三個數的中位數（中間大小的值）。

**輸入／輸出範例**
```
輸入：3 1 2
輸出：2
```

**提示**
中位數 = 三數的「最大值與最小值之外」的那一個；可巧妙組合 `Math.min` 與 `Math.max`。

**解題引導**
1. 觀察：中位數一定 ≥ 三數中的最小值，且 ≤ 三數中的最大值。
2. 經典公式：`Math.max(Math.min(a, b), Math.min(Math.max(a, b), c))`。
3. 若覺得公式難記，也可改用 if/else 逐一比較，或先排序三個數再取中間。

**參考解答**
```java
public class Q01 {
    public static int median(int a, int b, int c) {
        return Math.max(Math.min(a, b), Math.min(Math.max(a, b), c));
    }

    public static void main(String[] args) {
        System.out.println(median(3, 1, 2)); // 2
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| 3, 1, 2 | 2 |
| 5, 5, 5 | 5 |
| 10, 2, 8 | 8 |
| 1, 2, 3 | 2 |

**知識點說明**
`Math.min`／`Math.max` 一次只比較兩個值。中位數公式利用「夾擠」邏輯：取兩兩較大者中的較小、與較小者中的較大互相搭配。初學者也可先把三數放進陣列 `Arrays.sort` 後取 `a[1]`，但那是 O(n log n)；本公式為 O(1)。

## 第 2 題：階乘計算 ★

**題目**
輸入一個非負整數 `n`，回傳 `n!`（n 階乘，`0! = 1`）。

**輸入／輸出範例**
```
輸入：5
輸出：120
```

**提示**
用迴圈把 1 到 n 連乘；數字增長很快，回傳型別要用 `long`。

**解題引導**
1. 宣告 `long result = 1`（從 1 開始，因為 `0! = 1`）。
2. 用 `for` 從 2 連乘到 n。
3. 回傳 `result`。

**參考解答**
```java
public class Q02 {
    public static long factorial(int n) {
        long result = 1;
        for (int i = 2; i <= n; i++) {
            result *= i;
        }
        return result;
    }

    public static void main(String[] args) {
        System.out.println(factorial(5));  // 120
        System.out.println(factorial(0));  // 1
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| 0 | 1 |
| 1 | 1 |
| 5 | 120 |
| 10 | 3628800 |
| 20 | 2432902008176640000 |

**知識點說明**
階乘成長極快：`13!` 就超過 `int` 上限（約 21 億），`21!` 連 `long` 都會溢位。本題用 `long` 可安全到 20。迴圈從 2 開始是小優化（乘以 1 沒意義）。若需更大數值，要改用 `BigInteger`。

## 第 3 題：字元類型分類 ★

**題目**
輸入一個字元 `c`，判斷它屬於「數字」、「字母」、「空白」或「其他」，回傳對應字串。

**輸入／輸出範例**
```
輸入：A
輸出：字母
```

**提示**
善用 `Character` 類別內建的 `isDigit`、`isLetter`、`isWhitespace` 工具方法。

**解題引導**
1. 先檢查 `Character.isDigit(c)` → 回傳「數字」。
2. 再檢查 `Character.isLetter(c)` → 回傳「字母」。
3. 再檢查 `Character.isWhitespace(c)` → 回傳「空白」。
4. 都不符合 → 回傳「其他」。

**參考解答**
```java
public class Q03 {
    public static String classify(char c) {
        if (Character.isDigit(c)) return "數字";
        if (Character.isLetter(c)) return "字母";
        if (Character.isWhitespace(c)) return "空白";
        return "其他";
    }

    public static void main(String[] args) {
        System.out.println(classify('A')); // 字母
        System.out.println(classify('7')); // 數字
        System.out.println(classify(' ')); // 空白
        System.out.println(classify('@')); // 其他
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| A | 字母 |
| 7 | 數字 |
| （空白） | 空白 |
| @ | 其他 |

**知識點說明**
`char` 在 Java 底層是 16 位元的 Unicode 編碼，可當整數運算。雖然能用 `c >= '0' && c <= '9'` 手動判斷，但 `Character` 的工具方法更可靠（也支援全形數字等 Unicode 字元）。判斷順序要從最明確的條件開始。

## 第 4 題：矩陣對角線和 ★★

**題目**
給定一個 `n × n` 的二維整數陣列，回傳「主對角線」與「副對角線」上所有元素的總和；若 n 為奇數，正中央的元素只計算一次。

**輸入／輸出範例**
```
輸入：
1 2 3
4 5 6
7 8 9
輸出：25   （主對角線 1+5+9=15，副對角線 3+7=10，中央 5 不重複）
```

**提示**
主對角線元素是 `m[i][i]`，副對角線元素是 `m[i][n-1-i]`；當 `i == n-1-i` 時兩者重疊。

**解題引導**
1. 取得 `n = m.length`。
2. 用單層迴圈 `i` 從 0 到 n-1，累加主對角線 `m[i][i]`。
3. 同一圈內，若 `i != n-1-i` 才再累加副對角線 `m[i][n-1-i]`（避免中央重複）。
4. 回傳總和。

**參考解答**
```java
public class Q04 {
    public static int diagonalSum(int[][] m) {
        int n = m.length;
        int sum = 0;
        for (int i = 0; i < n; i++) {
            sum += m[i][i];               // 主對角線
            if (i != n - 1 - i) {
                sum += m[i][n - 1 - i];   // 副對角線（中央不重複）
            }
        }
        return sum;
    }

    public static void main(String[] args) {
        int[][] m = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        System.out.println(diagonalSum(m)); // 25
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| {{1,2,3},{4,5,6},{7,8,9}} | 25 |
| {{1,2},{3,4}} | 10 |
| {{5}} | 5 |
| {{1,1,1},{1,1,1},{1,1,1}} | 5 |

**知識點說明**
二維陣列在 Java 是「陣列的陣列」，`m.length` 是列數、`m[i].length` 是第 i 列的行數。副對角線索引 `n-1-i` 是常見的「鏡射」寫法。中央重疊的判斷（`i == n-1-i`）是本題最易被忽略的陷阱。

## 第 5 題：計算母音數量 ★★

**題目**
輸入一個字串，計算其中英文母音（a、e、i、o、u，不分大小寫）的總數。

**輸入／輸出範例**
```
輸入：Hello
輸出：2
```

**提示**
把字串轉小寫後逐字檢查；用 `"aeiou".indexOf(c)` 判斷是否為母音很方便。

**解題引導**
1. 先 `toLowerCase()` 統一大小寫。
2. 用 `toCharArray()` 或 `charAt` 逐字走訪。
3. 對每個字元，若 `"aeiou".indexOf(c) >= 0` 就計數加一。
4. 回傳計數。

**參考解答**
```java
public class Q05 {
    public static int countVowels(String s) {
        int count = 0;
        for (char c : s.toLowerCase().toCharArray()) {
            if ("aeiou".indexOf(c) >= 0) {
                count++;
            }
        }
        return count;
    }

    public static void main(String[] args) {
        System.out.println(countVowels("Hello")); // 2
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| Hello | 2 |
| AEIOU | 5 |
| xyz | 0 |
| Java | 2 |

**知識點說明**
`String.indexOf(char)` 找不到時回傳 -1，找到則回傳索引（≥ 0），常用來做「是否屬於某集合」的簡潔判斷。`toCharArray()` 把字串拆成 `char[]` 方便用增強型 for 走訪。記得先轉小寫，否則大寫母音會漏算。

## 第 6 題：十進位轉二進位 ★★

**題目**
輸入一個非負整數 `n`，回傳其二進位表示法的字串（不含前導 0；`0` 本身回傳 "0"）。

**輸入／輸出範例**
```
輸入：10
輸出：1010
```

**提示**
反覆「除以 2 取餘數」，餘數由下往上組成答案；用 `StringBuilder` 最後反轉。

**解題引導**
1. 特例：`n == 0` 直接回傳 "0"。
2. 當 `n > 0`：把 `n % 2` 接到 `StringBuilder`，再 `n /= 2`。
3. 迴圈結束後，因為餘數是反序產生的，要 `reverse()` 再轉字串。

**參考解答**
```java
public class Q06 {
    public static String toBinary(int n) {
        if (n == 0) return "0";
        StringBuilder sb = new StringBuilder();
        while (n > 0) {
            sb.append(n % 2);
            n /= 2;
        }
        return sb.reverse().toString();
    }

    public static void main(String[] args) {
        System.out.println(toBinary(10));  // 1010
        System.out.println(toBinary(0));   // 0
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| 0 | 0 |
| 5 | 101 |
| 10 | 1010 |
| 255 | 11111111 |

**知識點說明**
「除 2 取餘」是進制轉換的通法：餘數序列反過來就是目標進位。內建的 `Integer.toBinaryString(n)` 也能直接得到結果，但自己實作能理解原理。注意若忘了處理 `n == 0`，迴圈不會執行，會回傳空字串。

## 第 7 題：動物叫聲（介面與預設方法）★★★

**題目**
定義介面 `Animal`，含抽象方法 `sound()`（回傳叫聲）與「預設方法」`describe()`（回傳「這隻動物說：<叫聲>」）。實作 `Dog`（汪汪）與 `Cat`（喵喵），放進同一陣列並印出各自的 `describe()`。

**輸入／輸出範例**
```
Dog -> 這隻動物說：汪汪
Cat -> 這隻動物說：喵喵
```

**提示**
介面中可用 `default` 關鍵字提供有實作的方法，子類別不必覆寫即可直接使用。

**解題引導**
1. `interface Animal`：宣告 `String sound();`，並用 `default String describe()` 提供共用實作。
2. `Dog implements Animal`、`Cat implements Animal`，各自實作 `sound()`。
3. 建立 `Animal[]` 陣列同時放入 Dog 與 Cat。
4. 走訪呼叫 `describe()`，會自動串到各子類別的 `sound()`（多型）。

**參考解答**
```java
public class Q07 {
    interface Animal {
        String sound();
        default String describe() {
            return "這隻動物說：" + sound();
        }
    }

    static class Dog implements Animal {
        @Override
        public String sound() { return "汪汪"; }
    }

    static class Cat implements Animal {
        @Override
        public String sound() { return "喵喵"; }
    }

    public static void main(String[] args) {
        Animal[] animals = { new Dog(), new Cat() };
        for (Animal a : animals) {
            System.out.println(a.describe());
        }
    }
}
```

**測試案例**
| 物件 | 預期輸出 |
|------|---------|
| Dog | 這隻動物說：汪汪 |
| Cat | 這隻動物說：喵喵 |

**知識點說明**
Java 8 起，介面可有 `default` 方法（含實作），讓介面在演進時不必強迫所有實作類別改寫。`describe()` 內呼叫 `sound()`，實際執行哪個版本由物件真實型別決定（動態繫結）。介面與抽象類別的差異：一個類別可實作「多個」介面，但只能繼承「一個」類別。

## 第 8 題：括號配對驗證 ★★★

**題目**
給一個只含 `()`、`[]`、`{}` 的字串，判斷括號是否「正確配對且正確巢狀」。

**輸入／輸出範例**
```
輸入：{[]}
輸出：true

輸入：([)]
輸出：false
```

**提示**
用堆疊（後進先出）：遇左括號就壓入，遇右括號就彈出比對是否相配。

**解題引導**
1. 用 `ArrayDeque<Character>` 當堆疊。
2. 走訪字元：是左括號 `( [ {` → `push`。
3. 是右括號 → 若堆疊空則 false；否則 `pop` 出棧頂，檢查是否與當前右括號相配。
4. 走訪結束後，堆疊必須為空才代表全部配對成功。

**參考解答**
```java
import java.util.ArrayDeque;
import java.util.Deque;

public class Q08 {
    public static boolean isValid(String s) {
        Deque<Character> stack = new ArrayDeque<>();
        for (char c : s.toCharArray()) {
            if (c == '(' || c == '[' || c == '{') {
                stack.push(c);
            } else {
                if (stack.isEmpty()) return false;
                char top = stack.pop();
                if (c == ')' && top != '(') return false;
                if (c == ']' && top != '[') return false;
                if (c == '}' && top != '{') return false;
            }
        }
        return stack.isEmpty();
    }

    public static void main(String[] args) {
        System.out.println(isValid("{[]}"));  // true
        System.out.println(isValid("([)]"));  // false
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| () | true |
| ()[]{} | true |
| (] | false |
| ([)] | false |
| {[]} | true |

**知識點說明**
括號配對是堆疊的經典應用：最近開啟的括號必須最先關閉，正好符合 LIFO（後進先出）。官方建議用 `ArrayDeque` 取代舊的 `Stack` 類別（`Stack` 繼承自 `Vector`，有同步開銷且設計過時）。別忘了最後檢查堆疊是否清空——`"((" ` 這種只開不關的情況就靠這一步抓出。

## 第 9 題：解析整數清單（容錯）★★★

**題目**
給一個字串陣列，逐一嘗試轉成整數；能成功轉換的保留其值，無法轉換（非數字）的以 `0` 取代，回傳整數清單。

**輸入／輸出範例**
```
輸入：["1", "2", "x", "4"]
輸出：[1, 2, 0, 4]
```

**提示**
`Integer.parseInt` 遇到非數字會丟出 `NumberFormatException`，用 try-catch 攔截。

**解題引導**
1. 建立 `List<Integer>` 收集結果。
2. 對每個字串，先 `trim()` 去除前後空白。
3. 在 try 區塊呼叫 `Integer.parseInt`，成功就加入該值。
4. catch `NumberFormatException` 時改加入 `0`。

**參考解答**
```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Q09 {
    public static List<Integer> parseInts(String[] tokens) {
        List<Integer> result = new ArrayList<>();
        for (String t : tokens) {
            try {
                result.add(Integer.parseInt(t.trim()));
            } catch (NumberFormatException e) {
                result.add(0);
            }
        }
        return result;
    }

    public static void main(String[] args) {
        String[] input = {"1", "2", "x", "4"};
        System.out.println(parseInts(input)); // [1, 2, 0, 4]
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| ["1","2","x","4"] | [1, 2, 0, 4] |
| ["10","20"] | [10, 20] |
| ["a","b"] | [0, 0] |
| [" 5 ","-3"] | [5, -3] |

**知識點說明**
`NumberFormatException` 是非受檢例外（繼承 `RuntimeException`），可不強制處理，但在解析外部輸入時用 try-catch 容錯是良好實務。把 try-catch 放在迴圈「內部」，可讓單一壞值不中斷整批處理。`trim()` 能避免前後空白造成的誤判。

## 第 10 題：氣泡排序 ★★★

**題目**
自行實作氣泡排序，將整數陣列「由小到大」原地排序（不可使用 `Arrays.sort`）。

**輸入／輸出範例**
```
輸入：[5, 2, 4, 1, 3]
輸出：[1, 2, 3, 4, 5]
```

**提示**
重複走訪，把相鄰且順序錯誤的兩數交換；每一輪最大值會「浮」到尾端。

**解題引導**
1. 外層 `i` 控制輪數（共 n-1 輪）。
2. 內層 `j` 從頭比較相鄰 `a[j]` 與 `a[j+1]`，較大者往後換。
3. 每輪內層範圍可縮短（尾端已排好），用 `n-1-i`。
4. 進階優化：若某一輪沒有發生任何交換，代表已排序完成，可提早 `break`。

**參考解答**
```java
import java.util.Arrays;

public class Q10 {
    public static void bubbleSort(int[] a) {
        int n = a.length;
        for (int i = 0; i < n - 1; i++) {
            boolean swapped = false;
            for (int j = 0; j < n - 1 - i; j++) {
                if (a[j] > a[j + 1]) {
                    int tmp = a[j];
                    a[j] = a[j + 1];
                    a[j + 1] = tmp;
                    swapped = true;
                }
            }
            if (!swapped) break; // 已排好，提早結束
        }
    }

    public static void main(String[] args) {
        int[] a = {5, 2, 4, 1, 3};
        bubbleSort(a);
        System.out.println(Arrays.toString(a)); // [1, 2, 3, 4, 5]
    }
}
```

**測試案例**
| 輸入 | 預期輸出 |
|------|---------|
| [5,2,4,1,3] | [1, 2, 3, 4, 5] |
| [1,2,3] | [1, 2, 3] |
| [3,2,1] | [1, 2, 3] |

**知識點說明**
